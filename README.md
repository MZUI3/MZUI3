## 🖥️ Projects

### 🤖 리뷰 분석 Agent

> 기간 : 2026.05
> 역할 : DB 관리, Agent 설계 및 구현
>
> KT AIVLE School 미니 프로젝트

#### 문제 해결 경험

초기 설계에서는 Supervisor 패턴만으로 Agent를 구성하여 부정 리뷰에 대한 맞춤형 해결 방안을 생성하기 어려웠습니다.

기능 축소 의견이 제시되었지만, 직접 Router 기반 Agent 구조를 설계하여 리뷰 유형에 따라 적절한 분석 경로를 선택하도록 개선했습니다.

이를 통해 부정 리뷰 원인 분석 및 해결 방안 제시 기능을 완성했습니다.

### 📚 AI 도서 관리 시스템

> 기간 : 2026.05
>
> 역할 : 도서 등록 기능 구현, DB 관리, 알라딘 API 연동, AI 표지 생성 기능 개선
>
> KT AIVLE School 미니 프로젝트

#### 문제 해결 경험

기존 방식은 원하는 표지 이미지가 생성될 때까지 반복 생성하는 구조여서 토큰 사용량이 과도하게 발생했습니다.

사용자가 직접 콘티 이미지를 입력하도록 설계하고, 이를 분석하여 프롬프트에 반영하는 방식을 도입했습니다.

그 결과 반복 생성 횟수를 크게 줄여 토큰 사용량을 약 90% 절감했습니다.

#### 기능 구현 경험

알라딘 Open API를 연동하여 책 제목 검색 시 후보 목록을 제공하고, 선택한 도서의 제목·저자·설명을 자동 입력하도록 구현했습니다.

이를 통해 도서 등록 과정을 단순화하고 사용자 편의성을 향상시켰습니다.
